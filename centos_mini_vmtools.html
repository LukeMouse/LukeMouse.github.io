CentOS 6.5 minimal 安装配置VMware tools
  • 使用root账户登录到系统。
  • 配置网络，编辑/etc/sysconfig/network-script/ifcfg-eth0，修改NM_CONTROLLED取值为"no"【不依赖NetworkManager控制】，ONBOOT取值为"yes"【设置开机启动】。
  • 安装perl：yum -y install perl
  • 在VMware菜单中点击“安装VMware Tools(T)...
  • 执行如下命令进行安装 
    mkdir /media/mnt
mount /dev/cdrom /media/mnt/
cp VMwareTools-<XXX>.tar.gz /root
umount /dev/cdrom
cd /root
tar -zxvf VMwareTools-<XXX>.tar.gz
cd vmware-tools-distrib
./vmware-install.pl -d
