python环境配置

CentOS下安装pip

pip是python的包安装和管理工具，在CentOS下安装需要配置EPEL仓库，配置方法如下： CentOS6 

rpm -iUvh http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm

CentOS7 

rpm -iUvh http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-5.noarch.rpm

然后运行：yum clean all ; yum -y update和yum -y install python-pip 即可完成安装

升级python到2.7后重新安装pip

  1. 卸载原有的pip：yum -y erase python-pip
  2. 使用最新版本的python执行：python distribute_setup.py distribute_setup.py脚本
  3. 执行/usr/local/bin/easy_install pip 重新安装pip
  4. 使用硬连接将pip放到path路径：ln -s /usr/local/bin/pip /usr/bin/pip
