Centos Git1.7.1升级到Git2.2.1
作者
http://www.tuicool.com/articles/Y3eqIf

安装需求：

># yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel asciidoc
># yum install  gcc perl-ExtUtils-MakeMaker   
># wget http://ftp.gnu.org/pub/gnu/libiconv/libiconv-1.14.tar.gz
># tar zxvf libiconv-1.14.tar.gz 
># cd libiconv-1.14
># ./configure --prefix=/usr/local/libiconv
># make && make install

卸载Centos自带的git1.7.1:

通过git Cversion查看系统带的版本，Cento6.5应该自带的是git版本是1.7.1 

># yum remove git

下载git2.10.0并将git添加到环境变量中

># wget https://github.com/git/git/archive/v2.10.0.tar.gz
># tar zxvf v2.10.0.tar.gz
># cd git-2.10.0
># make configure
># ./configure --prefix=/usr/local/git --with-iconv=/usr/local/libiconv
># make all doc
># sudo make install install-doc install-html
># echo "export PATH=$PATH:/usr/local/git/bin" >> /etc/profile.d/git.sh
># source /etc/profile

查看版本号

># git --version
>git version 2.2.1

参考文章：http://www.dbpoo.com/centos-65-git-update/
