手动添加swap分区
作者
fromweb http://www.linuxidc.com/Linux/2015-02/113250.htm
  • 首先查看swap大小 
    # free
             total       used       free     shared    buffers     cached
Mem:       7961064    7796028     165036      90264     195104    3729528
-/+ buffers/cache:    3871396    4089668
Swap:      0          0    0
    显示为0说明交换分区不存在
  • 使用dd命令创建一个swap分区 
    #  dd if=/dev/zero of=/home/swap bs=1024 count=8388608
    这样就创建了一个8g的交换分区
  • 格式化新建的分区 
    #  mkswap /home/swap
  • 把新建的分区变成swap分区 
    #  swapon /doiido/swap
    注:关闭SWAP分区命令为：# swapoff /doiido/swap
  • 再次查看swap大小 
    #  free
             total       used       free     shared    buffers     cached
Mem:       7961064    7796028     165036      90264     195104    3729528
-/+ buffers/cache:    3871396    4089668
Swap:      8388604          0    8388604
  • 开机自动挂载swap 
    #  echo "/doiido/swap swap swap defaults  0 0" >> /etc/fstab
    这样SWAP分区就创建完毕
